# STS Analyser

A Streamlit web app for analysing STS XML test run files.

## Run locally

```bash
# One-time setup
brew install pipx && pipx ensurepath && pipx install streamlit

# Run the app
streamlit run app.py
```

## Deploy

Deploy for free on [Streamlit Community Cloud](https://share.streamlit.io) — point it at `app.py` in this repo and share the URL.
